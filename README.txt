
This website was developed using Flask on Python 2.7.3

Software to download:

- Python 2.7.3 (probably preinstalled)
- pip (from get-pip.py script, which can be found online)
- flask (from pip)
- PostgreSQL (apt-get install postgresql)
- Eclipse Luna and Java 7 for development
- PyDev for Eclipse (can be found on Eclipse Marketplace)
